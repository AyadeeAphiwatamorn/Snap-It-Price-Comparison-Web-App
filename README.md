# Snap-It-Price-Comparison-Web-App
Snap It is a UI/UX design project for a price comparison platform that helps users compare product prices across different stores and vendors. The project focuses on responsive layouts, accessibility, and modern user-centered interface design.


## Features
- Product price comparison interface
- Responsive web and mobile-friendly layouts
- Modern dark-themed UI design
- Clean visual hierarchy and navigation flow
- Accessibility-focused user experience
- Interactive and user-friendly interface design

## Tools & Technologies
- Figma
- UI/UX Design
- Responsive Design
- Wireframing
- Prototyping

## Project Focus
The complete structure, UI/UX workflow, layouts, user flow, branding, and interface elements were independently conceptualized and designed by myself. The project focused on improving usability and simplifying product comparison through clean and intuitive user experiences.
